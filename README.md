# Лендинг «OmarPrint.com»

Стек: Vite + React + TailwindCSS.

## Как запустить

1. Убедитесь, что интернет доступен и переменная `npm_config_offline` не принудительно `true`.
2. Установите зависимости:
   ```sh
   npm install
   ```
3. Запустите dev-сервер:
   ```sh
   npm run dev
   ```
   По умолчанию проект откроется на `http://localhost:5173`.
